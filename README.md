# Amazon Sales Performance Analysis

## Project Overview

This Power BI project analyzes 100,000 Amazon E-Commerce transactions to evaluate:

- Revenue Performance
- Customer Retention
- Category Performance
- Brand Performance
- Operational Health
- Discount Impact

---

## Dataset Information

- Rows: 100,000
- Columns: 25
- Tool Used: Power BI

---

## Key KPIs

| KPI | Value |
|------|--------|
| Revenue | ₹91.02M |
| Net Revenue | ₹84.24M |
| Orders | 100K |
| Customers | 43K |
| AOV | ₹842.37 |
| Repeat Customers | 68.7% |

---

## Key Insights

### Customer Retention

- Repeat Customers: 68.7%
- One-Time Customers: 31.3%

### Revenue Drivers

- Electronics generated highest revenue
- Revenue distribution is balanced across categories

### Brand Analysis

- CoreTech generated highest revenue
- UrbanStyle recorded highest AOV

### Operational Health

- Delivered Orders: 74.63%
- Returned Orders: 3.05%
- Cancelled Orders: 3.03%

### Discount Analysis

- Home & Kitchen received highest discounts
- Higher discounts do not always generate highest revenue
